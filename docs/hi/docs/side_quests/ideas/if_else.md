## 1. गाय को प्रसिद्ध वैज्ञानिकों के उद्धरण बोलवाएं

इस सेक्शन में कुछ स्ट्रेच एक्सरसाइज़ हैं, जो तुमने अब तक जो सीखा है उसका अभ्यास करने के लिए हैं।
ये एक्सरसाइज़ करना ट्रेनिंग के बाद के भागों को समझने के लिए _आवश्यक नहीं_ है, लेकिन गाय को प्रसिद्ध वैज्ञानिकों के उद्धरण बोलवाने का तरीका समझकर अपनी सीख को मजबूत करने का एक मज़ेदार तरीका प्रदान करती हैं।

### 1.1. `hello-containers.nf` स्क्रिप्ट को getQuote प्रोसेस का उपयोग करने के लिए संशोधित करें

हमारे पास `containers/data/pioneers.csv` फ़ाइल में कंप्यूटर और जीव विज्ञान के अग्रदूतों की एक सूची है।
उच्च स्तर पर, इस एक्सरसाइज़ को पूरा करने के लिए तुम्हें यह करना होगा:

- डिफ़ॉल्ट `params.input_file` को `pioneers.csv` फ़ाइल की ओर इंगित करने के लिए संशोधित करें।
- एक `getQuote` प्रोसेस बनाएं जो प्रत्येक इनपुट के लिए उद्धरण प्राप्त करने के लिए `quote` कंटेनर का उपयोग करे।
- `getQuote` प्रोसेस के आउटपुट को `cowsay` प्रोसेस से कनेक्ट करें ताकि उद्धरण प्रदर्शित हो सके।

`quote` कंटेनर इमेज के लिए, तुम या तो पिछली स्ट्रेच एक्सरसाइज़ में खुद बनाई गई इमेज का उपयोग कर सकते हो या Seqera Containers से प्राप्त इमेज का उपयोग कर सकते हो।

!!! Hint "संकेत"

    तुम्हारे getQuote प्रोसेस के `script` ब्लॉक के लिए एक अच्छा विकल्प यह हो सकता है:
        ```groovy
        script:
            def safe_author = author.tokenize(' ').join('-')
            """
            quote "$author" > quote-${safe_author}.txt
            echo "-${author}" >> quote-${safe_author}.txt
            """
        ```

तुम इस एक्सरसाइज़ का समाधान `containers/solutions/hello-containers-4.1.nf` में पा सकते हो।

### 1.2. अपनी Nextflow पाइपलाइन को `quote` और `sayHello` मोड में एक्सीक्यूट करने की अनुमति देने के लिए संशोधित करें।

अपनी पाइपलाइन में कुछ ब्रांचिंग लॉजिक जोड़ें ताकि यह `quote` और `sayHello` दोनों के लिए इच्छित इनपुट स्वीकार कर सके।
यहाँ Nextflow वर्कफ़्लो में `if` स्टेटमेंट का उपयोग करने का एक उदाहरण है:

```groovy title="hello-containers.nf"
workflow {
    if (params.quote) {
        ...
    }
    else {
        ...
    }
    cowSay(text_ch)
}
```

!!! Hint "संकेत"

    तुम किसी प्रोसेस के आउटपुट चैनल को एक नाम असाइन करने के लिए `new_ch = processName.out` का उपयोग कर सकते हो।

तुम इस एक्सरसाइज़ का समाधान `containers/solutions/hello-containers-4.2.nf` में पा सकते हो।
